## Установка

Для работы с проектом необходимо иметь установленный poetry, причем путь до него должен быть прописан в PATH. 
Выполните команду:
```console
poetry install
```

## Переменные окружения

Скопируйте файл `.env.sample` в `.env` и заполните недостающие поля.
